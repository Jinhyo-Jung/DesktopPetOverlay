# Privacy Policy Baseline

## Allowed Metrics
1. Active foreground minutes.
2. Input event count per interval.
3. Optional app category time buckets.

## Disallowed Collection
1. Raw key content.
2. Clipboard payload.
3. Full window titles or document names by default.

## Controls
1. User opt-in or explicit setting toggle.
2. User reset for accumulated activity-derived EXP.
3. Transparent explanation of what is measured.
