# 개발 환경 구성

## 결론
이 프로젝트는 `WSL2(Ubuntu) + Node.js LTS + Electron Forge(Vite + TypeScript)` 조합으로 개발한다.
Windows 배포 목표(`.exe`)를 유지하면서, 오버레이(투명/항상 위/드래그/클릭 상호작용) 기능을 안정적으로 구현하기 위한 기준 환경이다.

## 필수 구성
1. 운영체제
- Windows 10/11
- WSL2 + Ubuntu 24.04

2. 런타임/도구
- Node.js LTS (권장: 22.x 이상)
- npm
- Git

3. 앱 프레임워크
- Electron
- TypeScript
- Vite (Electron Forge plugin)

4. 빌드/배포
- electron-forge
- Windows 배포물: `.exe` (Squirrel 기반)

## 현재 디렉터리 초기 세팅 결과
1. Electron Forge 템플릿(`vite-typescript`) 적용
2. 오버레이 기본 창 옵션 반영
- `transparent`
- `alwaysOnTop`
- `frame: false`
3. 기본 렌더러 UI 및 드래그 가능 영역 설정
4. npm 의존성 설치 완료

## 실행 명령 (WSL에서 실행)
```bash
npm install
npm run start
npm run lint
npm run make
```
