# 프로젝트 아이디어 소개

## 결론
- 아이디어명: **내 책상 위 AI 펫, 하루를 함께 키우다**
- 한줄 설명: **PC 활동과 돌봄 행동을 성장 경험으로 바꾸고, AI 대화로 정서 교감을 더하는 데스크톱 오버레이 펫 앱**

## 상세
- 핵심 가치: 생산성 시간도 “함께 키우는 경험”으로 전환
- 차별점: 클릭 통과 오버레이 + 스탯 육성 + 성장 단계 + 활동 EXP + AI 대화
- 사용자는 데스크톱 위에 항상 떠 있는 펫을 돌보며, 실제 PC 사용 시간과 입력 활동을 성장 보상으로 연결해 몰입감을 얻는다.
- Feed/Clean/Play 같은 즉각적 상호작용과 단계 진화 시스템이 결합되어, 짧은 사용 시간에도 분명한 성취 루프를 만든다.
- 여기에 AI 대화를 더해 단순 상태 표시를 넘어 “말이 통하는 동반자” 경험을 제공한다.

## 예상 고객과 문제-해결 정의(최대 3개)
1. 재택/사무직 지식근로자
문제: 하루 대부분을 PC 앞에서 보내지만, 작업 시간이 단조롭고 성취가 체감되지 않아 집중이 쉽게 끊긴다.
해결: PC 활동을 EXP로 환산하고 펫 성장으로 시각화해, 업무 시간을 즉시 보상되는 루프로 전환한다.

2. 혼자 공부하는 학생·취준생
문제: 장시간 학습 중 정서적 피로와 외로움이 쌓이고, 작은 휴식 이후 다시 몰입하기 어렵다.
해결: Feed/Clean/Play와 AI 대화를 결합해 짧은 상호작용만으로 감정 환기와 학습 복귀 트리거를 제공한다.

3. 가벼운 게임형 습관 관리 선호 사용자
문제: 기존 할 일 앱은 숫자/텍스트 중심이라 재미가 부족해 꾸준히 사용하기 어렵다.
해결: 오버레이 펫의 단계 진화, 상태 변화, 일일 리포트로 습관 관리 과정을 게임처럼 체험하게 한다.
