# 제품 아이디어 소개 영상 대본

## 결론
- 총 길이: 3분
- 구성: 소개 1분 + 시연 2분
- 목표: “내 책상 위 AI 펫, 하루를 함께 키우다”의 가치와 실제 사용 흐름을 직관적으로 전달

## 0:00 ~ 1:00 (아이디어 소개 1분)
- `0:00 ~ 0:10`
화면: 바쁜 데스크톱 화면, 반복 작업 장면
내레이션: "하루 종일 PC 앞에 있는데, 내 시간은 어디로 갔을까요?"

- `0:10 ~ 0:25`
화면: 앱 타이틀 노출, 펫이 오버레이로 나타남
내레이션: "이 프로젝트는 내 책상 위 AI 펫, 하루를 함께 키우다입니다. PC 활동과 돌봄 행동을 성장 경험으로 바꾸는 데스크톱 오버레이 펫 앱입니다."

- `0:25 ~ 0:40`
화면: 핵심 기능 키워드 카드 순차 노출 (오버레이, 스탯, 진화, 활동 EXP, AI 대화)
내레이션: "클릭 통과 오버레이, 스탯 육성, 단계 진화, 활동 EXP, AI 대화를 한 화면에 담았습니다."

- `0:40 ~ 0:55`
화면: 직장인/학생/습관 관리 사용자 상황 컷
내레이션: "재택 근무자, 공부하는 학생, 재미있는 습관 관리를 원하는 사용자에게, 짧은 상호작용만으로도 성취 루프를 제공합니다."

- `0:55 ~ 1:00`
화면: “2분 실사용 시연” 자막 전환
내레이션: "지금부터 2분 동안 실제 동작을 보여드리겠습니다."

## 1:00 ~ 3:00 (시연 2분)
- `1:00 ~ 1:20`
화면: 앱 실행 -> 펫 표시 -> 패널 열기
내레이션: "앱을 실행하면 데스크톱 위에 펫이 항상 표시되고, 메인 캐릭터 클릭으로 패널을 열 수 있습니다."

- `1:20 ~ 1:40`
화면: Feed/Clean/Play 버튼 클릭, 스탯/EXP 즉시 반영
내레이션: "Feed, Clean, Play로 즉시 상호작용하고, 스탯과 EXP 변화가 바로 반영됩니다."

- `1:40 ~ 2:00`
화면: EXP 증가 후 단계 전환 메시지 노출
내레이션: "EXP가 누적되면 단계가 진화하고, 전환 이벤트 메시지로 성장 순간을 명확하게 보여줍니다."

- `2:00 ~ 2:20`
화면: 활동 EXP 영역 (자동 집계 + 수동 획득 버튼)
내레이션: "PC 활동은 자동으로 활동 EXP로 환산되고, 필요할 때 수동 획득으로 보완할 수 있습니다."

- `2:20 ~ 2:40`
화면: 대화하기 클릭 -> 톱니 버튼 -> API 키 재입력/삭제/저장 폴더 열기
내레이션: "대화하기에서는 AI와 짧은 교감이 가능하고, 톱니 메뉴에서 API 키 재입력, 삭제, 저장 폴더 확인까지 간단히 처리할 수 있습니다."

- `2:40 ~ 2:55`
화면: 일일 리포트 팝업 또는 R 버튼 열람 장면
내레이션: "하루 마무리에는 일일 요약과 격려 리포트로 오늘의 돌봄과 활동 성과를 확인합니다."

- `2:55 ~ 3:00`
화면: 앱 로고 + 한줄 메시지
내레이션: "내 책상 위 AI 펫과 함께, 오늘의 PC 시간을 성장의 기록으로 바꿔보세요."
