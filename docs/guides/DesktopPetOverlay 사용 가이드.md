# DesktopPetOverlay 사용 가이드

## 결론
- 메인 캐릭터를 클릭하면 설정 UI가 열리고, 다시 클릭하면 닫힙니다.
- `활동 EXP 기록 초기화` 버튼은 제거되었습니다.
- `EXP 획득` 버튼은 항상 같은 라벨로 표시되며, 쿨다운 중 클릭하면 "몇 초 뒤 재클릭 가능한지" 안내합니다.

## 빠른 시작
1. 실행 파일
- `out/DesktopPetOverlay-win32-x64/DesktopPetOverlay.exe`

2. 기본 토글
- 메인 캐릭터 클릭: UI 열기/닫기
- `Ctrl+Alt+Shift+O`: 클릭 통과 ON/OFF

## 주요 버튼
1. `Feed`, `Clean`, `Play`
- 실제 스탯이 회복될 때만 EXP를 획득합니다.
- 스탯이 이미 최대치라 변화가 없으면 EXP를 주지 않습니다.

2. `활동 EXP: ON/OFF`
- 자동 활동 EXP 수집을 켜고 끕니다.

3. `EXP 획득`
- 수동 EXP(+2)를 받는 버튼입니다.
- 5분 쿨다운 중에는 버튼 클릭 시 남은 초를 안내합니다.

4. `Help`
- 각 버튼 의미와 EXP 획득 규칙을 앱 내에서 확인할 수 있습니다.

## EXP 확인
1. `EXP` 숫자 클릭
- `keydown`, `mousedown`, `mousemove`, `wheel`, `touchstart` 집계를 펼쳐서 확인합니다.

2. 활동 EXP 의미
- `활동 EXP(자동)`: 자동 수집으로 얻은 EXP
- `EXP 획득(수동)`: 버튼 클릭으로 얻은 EXP

## 오버레이 동작
1. 기본 배치
- 창은 기본적으로 작업 표시줄 위 영역(화면 하단 우측)에서 시작합니다.

2. 캐릭터 이동
- 캐릭터를 드래그하여 위치를 이동할 수 있습니다.
- 종료 후 재실행 시 마지막 위치를 복원합니다.

## 문제 해결
1. UI가 클릭되지 않을 때
- 클릭 통과가 ON일 수 있습니다. `Ctrl+Alt+Shift+O`로 OFF 후 다시 시도하세요.

2. 활동 EXP가 바로 오르지 않을 때
- 자동 활동 EXP는 샘플 주기(5분) 단위로 반영됩니다.
- EXP 수치를 눌러 집계 상세를 확인하세요.
