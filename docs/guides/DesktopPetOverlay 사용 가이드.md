# DesktopPetOverlay 사용 가이드

## 결론
- 메인 캐릭터는 작업표시줄 상단 라인을 따라 이동하며, 화면 경계를 넘어 다른 모니터로도 옮길 수 있습니다.
- 스탯(설정) 패널은 상단 `패널 이동` 핸들을 드래그해서 원하는 위치로 옮길 수 있습니다.
- 메인 캐릭터와 패널 모두 기존 바운딩 박스 표시 없이 동작합니다.

## 기본 조작
1. 메인 캐릭터 클릭
- 설정 UI 열기/닫기

2. 메인 캐릭터 드래그
- 작업표시줄 상단 라인을 따라 이동
- 모니터 경계를 넘어 다른 모니터로 이동 가능

3. `Ctrl+Alt+Shift+O`
- 클릭 통과 ON/OFF 전환

4. `ESC`
- 열린 설정 UI를 즉시 닫기

## 스탯 패널 이동
1. 이동 방법
- 설정 UI 상단 `⋮⋮ 패널 이동` 영역을 드래그합니다.

2. 동작 범위
- 패널은 현재 오버레이 창 내부에서 자유롭게 이동합니다.
- 이동한 위치는 저장되며 다음 실행 시 복원됩니다.

## 캐릭터 크기
1. 메인/보조 캐릭터
- 기존 대비 2배 크기(88px)로 표시됩니다.

## EXP / 버튼
1. `활동 EXP: ON/OFF`
- 자동 활동 EXP 수집 토글

2. `EXP 획득`
- 수동 EXP(+2) 획득 버튼
- 쿨다운 중 클릭하면 남은 초 안내

3. `Feed`, `Clean`, `Play`
- 실제 스탯이 회복될 때만 EXP 획득

## 문제 해결
1. UI가 클릭되지 않을 때
- 클릭 통과가 ON일 수 있으므로 `Ctrl+Alt+Shift+O`로 OFF 전환

2. 설정 UI가 캐릭터를 가려 닫기 어려울 때
- `ESC` 키를 눌러 닫기

3. 캐릭터 시작 위치를 기본값으로 되돌리고 싶을 때
- `%APPDATA%\DesktopPetOverlay\overlay-preferences.json`과 캐릭터 저장값을 초기화 후 재실행
